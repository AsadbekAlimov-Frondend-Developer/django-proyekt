# Oasis Ceramics - Дипломный Проект

Это проект веб-сайта для магазина посуды "Oasis Ceramics", разработанный на фреймворке Flask (Python).
Этот документ поможет вам запустить проект и понять его структуру для написания теоретической части диплома.

## 1. Структура проекта (Для описания в дипломе)
Проект построен с использованием архитектуры **MVC (Model-View-Controller)**, слегка адаптированной под Flask (M-T-V: Model-Template-View):

- **app.py** (Главный файл) - точка входа в приложение, подключение базы данных SQLite и конфигурация сессий для корзины.
- **models.py** (Model) - описание таблиц базы данных. Две таблицы: `Category` (Категории посуды) и `Product` (Товары).
- **routes.py** (Controller / View в терминологии Flask) - обработчик ссылок (маршрутов). Здесь описана логика Главной страницы, Каталога, добавления товаров в Корзину. 
- **admin_setup.py** - конфигурация админ-панели (с использованием Flask-Admin), которая позволяет легко управлять товарами без вмешательства в базу данных вручную.
- **templates/** (View / Templates) - папка с HTML-шаблонами. Мы используем шаблонизатор Jinja2, который позволяет наследовать общий каркас (`layout.html`) на все страницы сайта (`index.html`, `catalog.html`, `cart.html`).
- **static/css** - папка со статическими файлами. Чистый CSS написан с нуля без использования конструкторов, как и требовалось по заданию.

## 2. Как запустить проект локально (На вашем Mac/ПК)

Так как у вас изначально не установлен Python, вот простая инструкция:

### Шаг 1: Установка Python
1. Перейдите на официальный сайт Python: https://www.python.org/downloads/
2. Скачайте и установите последнюю версию для MacOS.
3. Откройте терминал (Terminal) и убедитесь, что установка прошла успешно, введя `python3 --version`.

### Шаг 2: Установка зависимостей проекта
1. Откройте терминал.
2. Перейдите в папку с проектом. Введите команду `cd ~/Desktop/test` и нажмите Enter.
3. Установите необходимые библиотеки с помощью команды (убедитесь, что вы находитесь в папке проекта):
```bash
pip3 install -r requirements.txt
```

### Шаг 3: Запуск сервера
1. В том же терминале введите команду:
```bash
python3 app.py
```
2. Если вы видите текст `Running on http://127.0.0.1:5000`, значит, сервер успешно запущен!
3. Откройте ваш браузер (Chrome/Safari) и перейдите по адресу: **http://127.0.0.1:5000**
4. Для входа в панель администратора перейдите по адресу: **http://127.0.0.1:5000/admin**
*(При первом запуске автоматически создастся файл `store.db` - это ваша база данных SQLite)*

## 3. Требования перед выгрузкой на сервер и защитой

Зайдите в `http://127.0.0.1:5000/admin` и добавьте несколько категорий (например, "Тарелки", "Кружки", "Чайные сервизы").
Затем добавьте несколько товаров, указав им категорию, цену и ссылку на картинку (image_url можно взять из интернета для демонстрации, или загрузить локально в папку `static`).

## 4. Как загрузить на реальный сервер?

Для реального хостинга (сервера) обычно используется платформа типа PythonAnywhere, Heroku или любой VPS-сервер:
- При деплое (развертывании) код из `app.py` подключается через WSGI/Gunicorn.
- Команда `pip install -r requirements.txt` выполнится на сервере аналогичным образом.
- Фронтенд (дизайн) и логика не изменятся.
- Для дипломного проекта отлично подойдёт демонстрация через локальный сервер (localhost).
